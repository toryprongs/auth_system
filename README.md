# auth_system
Простой бэкенд для регистрации и логина пользователей на FastAPI с SQLite.

Это простой проект на Python с использованием FastAPI и SQLite.
Система позволяет:
1. Регистрировать пользователей
2. Авторизовываться
3. Просматривать свой профиль
4. Обновлять информацию о пользователе
5. Мягко удалять пользователя

Проект является чистым бэкендом, все действия выполняются через API эндпоинты.

Стек:
1. Python 3.10+
2. FastAPI - для создания API
3. SQLAlchemy - ORM для работы с базой данных
4. SQLite - база данных (файл database.db)
5. Passlib - для хеширования паролей
6. PyJWT - для создания JWT токенов

Запуск проекта:
1. Создать виртуальное окружение (рекомендуется, но не обязательно).
2. Установить зависимости:
pip install fastapi uvicorn sqlalchemy passlib[bcrypt] pyjwt pydantic
3. Запустить сервер: uvicorn main:app --reload
4. Перейти в браузере по адресу: http://127.0.0.1:8000/docs - здесь можно тестировать все эндпоинты через Swagger UI.

/register - регистрация пользователя   
/login - авторизация / получение токена 
/me - просмотр своего профиля        
/update - обновление информации          
/delete - мягкое удаление пользователя
